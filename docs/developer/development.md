# 🛠️ Development Guide

This guide describes how to run Hornbill locally for development.

## Running the Development Servers

To start the local database engine and BFF API development servers in watch mode:

```bash
make dev
```

This script runs the local processes on the host. Ensure you have installed the project dependencies using `bun install` first. For more information on contributing, see [CONTRIBUTING.md](../../CONTRIBUTING.md).
